What is This?

PYRO Reloaded is an interface built by MrFUNCT1ON, for holders of PYRO on ETH and TRON. Recently, you may have heard the news that the creator of the PYRO project has abandoned and jumped ship. Along with him, he took down the website at pyro.network - so this interface is to enable the holders of the PYRO token, to still manage their HODLs!


Join the Community

The Discord server is still active, and the code to this interface is Open Source, available on GitHub - check them both out!
